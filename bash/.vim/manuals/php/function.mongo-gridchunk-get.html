mongo_gridchunk_get

(PECL mongo >= 0.8.0)

mongo_gridchunk_get — Return a chunk of a given file

Description

resource mongo_gridchunk_get ( resource $file , int $num )

Return a chunk of a given file.

Parameters

file

The file from which to fetch the chunk.

num

The number of the chunk to fetch.

Return Values

Returns a Gridfs file chunk.

See Also

  • mongo_gridfile_chunk_num() - Returns the number of chunks a file is composed of
